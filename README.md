# MikeBot
A general Discord bot for my server

## Deploy
```bash
> cd cmd/mikebot && go build
> ./mikebot token=<token here>
```

## Commands

### !roll
Rolls a dice, IN PROGRESS

```bash
> !roll 1d20+2
$ Mike rolled ([13] + 2) = 15
```

### !slap
Slaps a user

```bash
> !slap a large lad
$ Mike slaps a large lad with a big fish.
```
